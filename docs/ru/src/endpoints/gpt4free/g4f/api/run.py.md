# Модуль запуска API g4f

## Обзор

Этот модуль содержит точку входа для запуска API g4f (GPT4Free). Он импортирует модуль `g4f.api` и запускает API с возможностью отладки.

## Подробнее

Этот файл предназначен для запуска API, предоставляемого библиотекой g4f. Он использует функцию `run_api` из модуля `g4f.api` для запуска сервера API. Параметр `debug=True` включает режим отладки, который полезен во время разработки и тестирования.

## Функции

### `__main__`

```python
if __name__ == "__main__":
    g4f.api.run_api(debug=True)
```

**Назначение**: Запускает API g4f при выполнении файла как основного скрипта.

**Как работает функция**:

1. Проверяет, является ли текущий файл основным запускаемым скриптом (`if __name__ == "__main__":`).
2. Если да, вызывает функцию `g4f.api.run_api(debug=True)` для запуска API.
