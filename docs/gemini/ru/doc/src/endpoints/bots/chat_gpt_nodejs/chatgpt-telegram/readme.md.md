# Голосовой помощник chatgpt - telegram

## Обзор

Данный файл содержит краткое описание проекта "Голосовой помощник chatgpt - telegram". 

## Подробнее

Проект реализует голосового помощника на основе модели ChatGPT, интегрированного в Telegram. 

- **Цель:**  Разработать бота, который может выполнять различные задачи, связанные с обработкой текста и кода, с помощью ChatGPT.

- **Функциональность:** 
    - Принимает голосовые сообщения от пользователей Telegram.
    - Преобразует голосовые сообщения в текст.
    - Передает текст модели ChatGPT для обработки.
    - Получает ответ от модели ChatGPT.
    - Возвращает ответ пользователю в текстовом формате.

## Как использовать

### Настройка

1. **Создайте бота в Telegram:**
    - Найдите бота @BotFather в Telegram.
    - Начните чат с ботом @BotFather.
    - Введите команду `/newbot` и следуйте инструкциям для создания нового бота.
    - Запишите токен доступа, который бот @BotFather предоставит вам.

2. **Настройте API-ключ ChatGPT:**
    - Зарегистрируйтесь на сайте OpenAI.
    - Получите API-ключ из личного кабинета OpenAI.

3. **Замените местозаполнители:**
    - В файле `index.js` замените `YOUR_TELEGRAM_BOT_TOKEN` на токен доступа вашего бота.
    - Замените `YOUR_CHATGPT_API_KEY` на API-ключ ChatGPT.

4. **Запустите бота:**
    - Запустите команду `node index.js` в терминале.

### Использование

- **Отправьте голосовое сообщение:** 
    - Найдите своего бота в Telegram.
    - Отправьте голосовое сообщение.
- **Получите ответ:**
    - Бот обработает ваше голосовое сообщение, отправит его модели ChatGPT и вернет ответ в текстовом формате.

## Примеры использования

- **Получить информацию:** 
    "Расскажи мне о истории искусственного интеллекта."
- **Попросить перевести текст:**
    "Переведи 'Hello, world!' на испанский."
- **Сгенерировать код:** 
    "Напиши функцию на Python, которая вычисляет факториал числа."

## Дополнительная информация

- **Модель ChatGPT:** В проекте используется модель ChatGPT от OpenAI.

## Примечания

- **Ограничения:** 
    - Модель ChatGPT может иметь ограничения по длине ответов и количеству запросов в секунду. 
    - Точность и качество ответов зависят от качества и доступности данных, используемых для обучения модели ChatGPT. 
- **Безопасность:** 
    - Храните API-ключи в безопасном месте и не делитесь ими с другими.
