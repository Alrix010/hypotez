# Модуль с примерами агентов TinyTroupe

## Обзор

Этот модуль содержит примеры создания и загрузки агентов для библиотеки TinyTroupe. Он предоставляет функции для создания различных персонажей, таких как архитектор, специалист по данным, врач и лингвист, используя как загрузку из файлов спецификаций, так и программное определение атрибутов.

## Подробнее

Модуль демонстрирует, как можно создавать агентов TinyPerson с использованием различных подходов: загрузка спецификаций из файлов и программное определение атрибутов. Примеры включают Оскара-архитектора, Лизу-специалиста по данным, Маркоса-врача и Лилу-лингвиста. Эти агенты могут быть использованы как есть или модифицированы для создания собственных агентов.

## Функции

### `create_oscar_the_architect`

**Назначение**: Создает агента "Оскар, архитектор", загружая его спецификацию из файла.

**Параметры**:
- Нет.

**Возвращает**:
- `TinyPerson`: Объект агента "Оскар, архитектор".

**Как работает функция**:
- Функция вызывает `load_example_agent_specification("Oscar")` для загрузки спецификации агента из файла.
- Затем использует `TinyPerson.load_specification()` для создания объекта агента на основе загруженной спецификации.

**Примеры**:
```python
oscar = create_oscar_the_architect()
print(oscar.name) # Вывод: Oscar
print(oscar.occupation['title']) # Вывод: Architect
```

### `create_oscar_the_architect_2`

**Назначение**: Создает агента "Оскар, архитектор" программно, определяя его атрибуты.

**Параметры**:
- Нет.

**Возвращает**:
- `TinyPerson`: Объект агента "Оскар, архитектор", созданный программно.

**Как работает функция**:
1. **Создание объекта агента**:
   - Создается экземпляр класса `TinyPerson` с именем "Oscar".
2. **Определение атрибутов**:
   - Атрибуты агента, такие как возраст, национальность, поведение, род занятий, личность, предпочтения, навыки и отношения, определяются с использованием метода `oscar.define()`.
   - Метод `oscar.define()` принимает два аргумента: имя атрибута и значение атрибута.
3. **Возврат агента**:
   - Функция возвращает созданный и настроенный объект `oscar`.

**Примеры**:
```python
oscar = create_oscar_the_architect_2()
print(oscar.name) # Вывод: Oscar
print(oscar.occupation['title']) # Вывод: Architect
```
