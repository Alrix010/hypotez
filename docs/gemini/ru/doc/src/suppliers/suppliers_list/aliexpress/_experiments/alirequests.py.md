# Модуль для экспериментов с запросами на AliExpress

## Обзор

Модуль `alirequests.py` предназначен для экспериментов с запросами на сайт AliExpress. В текущей версии он использует веб-драйвер для открытия страницы AliExpress.

## Подробнее

Этот модуль является частью проекта `hypotez` и находится в стадии экспериментов. Он демонстрирует использование веб-драйвера для автоматизации действий на сайте AliExpress, таких как открытие страницы.
