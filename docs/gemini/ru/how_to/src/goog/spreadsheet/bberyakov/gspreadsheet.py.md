## Как использовать класс `GSpreadsheet`
=========================================================================================

Описание
-------------------------
Класс `GSpreadsheet` предоставляет набор функций для работы с таблицами Google Sheets, используя API `gspread`.  Он позволяет создавать, открывать, изменять и получать доступ к данным в таблицах.

Шаги выполнения
-------------------------
1. **Инициализация объекта `GSpreadsheet`**:
   - Создайте экземпляр класса `GSpreadsheet`, передав в конструктор идентификатор таблицы (`s_id`) или ее название (`s_title`). Если ни `s_id`, ни `s_title` не указаны, создается новый объект без привязки к конкретной таблице.
   - Используйте атрибут `gsh` для доступа к объекту `Spreadsheet` (таблице) Google Sheets.

2. **Получение данных о таблицах проекта**:
   - Вызовите метод `get_project_spreadsheets_dict()`, чтобы получить словарь с описанием всех таблиц проекта.

3. **Открытие таблицы по названию**:
   - Используйте метод `get_by_title(sh_title)`, передав название таблицы в качестве аргумента.
   - Если таблица не существует, она будет создана, а учетная запись `d07708766@gmail.com` получит права редактирования (`role='writer'`).

4. **Открытие таблицы по идентификатору**:
   - Используйте метод `get_by_id(sh_id)`, передав идентификатор таблицы в качестве аргумента.
   - Этот метод возвращает объект `Spreadsheet` Google Sheets, представляющий открытую таблицу.

5. **Получение списка всех таблиц текущего аккаунта**:
   - Вызовите метод `get_all_spreadsheets_for_current_account()`, чтобы получить список всех таблиц Google Sheets, доступных в текущем аккаунте.


Пример использования
-------------------------

```python
from src.goog.spreadsheet.bberyakov.gspreadsheet import GSpreadsheet

# Создание объекта GSpreadsheet с привязкой к таблице по ее идентификатору
spreadsheet = GSpreadsheet(s_id='1ZcK74BCgWKVr4kODjPmSvjp5IyO0OxhXdbeHKWzLQiM')

# Получение списка всех таблиц проекта
spreadsheets_dict = spreadsheet.get_project_spreadsheets_dict()

# Открытие таблицы по названию
spreadsheet.get_by_title('My Spreadsheet')

# Открытие таблицы по идентификатору
spreadsheet_by_id = spreadsheet.get_by_id('1ZcK74BCgWKVr4kODjPmSvjp5IyO0OxhXdbeHKWzLQiM')

# Получение списка всех таблиц текущего аккаунта
all_spreadsheets = spreadsheet.get_all_spreadsheets_for_current_account()
```
