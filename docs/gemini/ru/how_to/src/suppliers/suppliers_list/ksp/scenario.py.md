## Как использовать блок кода `get_list_products_in_category`
=========================================================================================

### Описание
-------------------------
Блок кода `get_list_products_in_category` извлекает список URL-адресов товаров со страницы категории поставщика. Функция проверяет наличие элементов  `product_links`, которые указывают на товары, и собирает их в список. 
При необходимости пролистывает страницы категории и собирает новые URL-адреса товаров. 

### Шаги выполнения
-------------------------
1. **Инициализация**: Получает объект `Supplier`  и его атрибуты (драйвер `d:Driver` и локаторы `l: dict`).
2. **Обработка баннера**: Закрывает баннер, используя локатор `close_banner`.
3. **Скроллинг**: Прокручивает страницу.
4. **Сбор URL-адресов**:  Извлекает список URL-адресов товаров `list_products_in_category: List`  с помощью `d.execute_locator` и локатора `product_links`.
5. **Обработка пустого списка**: Если список `list_products_in_category` пуст, выводит предупреждение в лог и возвращает `None`.
6. **Пролистывание**: Пока текущий URL-адрес не совпадает с предыдущим, выполняется пролистывание страниц категории:
   - Вызывает функцию `paginator` для проверки возможности пролистывания.
   - Если пролистывание возможно, добавляет новый список URL-адресов товаров  `list_products_in_category`.
   - Если пролистывание невозможно, прекращает цикл.
7. **Формирование списка**: Преобразовывает список `list_products_in_category` в формат `List`  и выводит в лог количество найденных товаров.
8. **Возвращение списка**: Возвращает список `list_products_in_category`.

### Пример использования
-------------------------

```python
from src.suppliers.ksp.scenario import get_list_products_in_category
from src.suppliers.suppliers_list.ksp.ksp import KSP

# Создаем объект поставщика KSP
supplier = KSP()

# Получаем список URL-адресов товаров из категории
products_urls = get_list_products_in_category(supplier)

# Выводим список URL-адресов
print(products_urls)
```
