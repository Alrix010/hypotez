## Как использовать этот блок кода
=========================================================================================

### Описание
-------------------------
Этот блок кода инициализирует проект `hypotez` и настраивает необходимые зависимости для работы с поставщиком `EtzMaleh`. Он выполняет следующие действия:

- Добавляет корневой каталог проекта `hypotez` в `sys.path` для возможности импорта модулей из других частей проекта.
- Импортирует необходимые модули для работы с `EtzMaleh`, включая:
    - `Driver` для работы с браузером.
    - `Product`, `ProductFields`, `Category` для работы с товарами и категориями.
    - `StringFormatter`, `StringNormalizer` для форматирования и нормализации строк.
    - `pprint` для удобной печати данных в консоль.
    - `Product` из `PrestaShop` для работы с API PrestaShop.
    - `save_text_file` для сохранения текстовых файлов.

- Определяет функцию `start_supplier`, которая принимает параметры поставщика (`supplier_prefix` и `locale`) и возвращает объект `Supplier`.

### Шаги выполнения
-------------------------
1. **Настройка пути:**
    - Определяется корневой каталог проекта `hypotez` с помощью `os.getcwd()`.
    - Добавляется корневой каталог в `sys.path` для импорта модулей из других частей проекта.
2. **Импорт модулей:**
    - Импортируются необходимые модули для работы с поставщиком `EtzMaleh`.
3. **Определение функции `start_supplier`:**
    - Определяется функция `start_supplier`, которая принимает два параметра:
        - `supplier_prefix`: Префикс поставщика (например, `aliexpress`).
        - `locale`: Локаль для работы с данными.
    - Функция создает словарь `params` с параметрами поставщика.
    - Функция возвращает объект `Supplier`, созданный с помощью `params`.

### Пример использования
-------------------------

from src.suppliers.etzmaleh._experiments.JUPYTER_header import start_supplier

# Запускаем поставщика 'aliexpress' с локалью 'en'
supplier = start_supplier(supplier_prefix='aliexpress', locale='en')
