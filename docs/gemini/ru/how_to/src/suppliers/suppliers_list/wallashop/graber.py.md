## Как использовать блок кода `Graber`
=========================================================================================

Описание
-------------------------
Класс `Graber` собирает значения полей товара с сайта `wallashop.co.il`. Он наследует от родительского класса `Graber`, который предоставляет базовые функции обработки полей. Если требуется нестандартная обработка поля, функция перегружается в этом классе.

Шаги выполнения
-------------------------
1. **Инициализация класса `Graber`:** Создается экземпляр класса `Graber`, передавая в него объект `driver` (вебдрайвер) и `lang_index` (индекс языка).
2. **Установление префикса поставщика:** В конструкторе класса устанавливается префикс поставщика `supplier_prefix`, который в данном случае равен `"wallashop"`.
3. **Инициализация родительского класса `Graber`:** Вызывается конструктор родительского класса `Graber`, передавая ему префикс поставщика, вебдрайвер и индекс языка.
4. **Установка локации для декоратора `close_pop_up`:**  Переменная `Config.locator_for_decorator` устанавливается в `None`. Это означает, что декоратор `close_pop_up` не будет автоматически выполняться. Если необходимо, можно установить локатор, чтобы декоратор выполнялся перед отправкой запроса к вебдрайверу.

Пример использования
-------------------------

```python
from src.suppliers.wallashop.graber import Graber
from src.webdriver.driver import Driver
from src.webdriver.browsers import Chrome

# Инициализация вебдрайвера
driver = Driver(Chrome)

# Создание экземпляра класса Graber
graber = Graber(driver, lang_index=0)
```
