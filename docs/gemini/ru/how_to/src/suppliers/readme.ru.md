### **Инструкция по использованию класса `Supplier`**

=========================================================================================

Описание
-------------------------\
Класс `Supplier` является базовым классом для всех поставщиков информации в системе. Он определяет общий интерфейс и логику взаимодействия с различными источниками данных, такими как веб-сайты, документы и базы данных. Класс обеспечивает унифицированный подход к работе с данными от разных поставщиков, используя уникальные префиксы для каждого из них.

Шаги выполнения
-------------------------\
1. **Инициализация**: Клиент инициализирует класс `Supplier` с необходимыми параметрами конфигурации и аутентификации.
2. **Настройка**: Класс `Supplier` настраивает взаимодействие с конкретным поставщиком, используя его уникальный префикс и параметры доступа.
3. **Аутентификация**: При необходимости, класс выполняет аутентификацию на стороне поставщика для получения доступа к данным.
4. **Запуск сценариев**: Класс запускает определенные сценарии для извлечения и обработки данных из источника поставщика.
5. **Получение данных**: Класс извлекает данные, используя локаторы и преобразует их в унифицированный формат.
