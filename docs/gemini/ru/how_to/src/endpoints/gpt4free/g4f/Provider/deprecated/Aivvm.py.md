### **Как использовать этот блок кода**
=========================================================================================

Описание
-------------------------
Этот код определяет класс `Aivvm`, который является провайдером для взаимодействия с сервисом `chat.aivvm.com`. Он позволяет отправлять запросы к моделям, таким как `gpt-3.5-turbo` и `gpt-4`, с поддержкой потоковой передачи ответов.

Шаги выполнения
-------------------------
1. **Импорт необходимых модулей**: Импортируются модули `requests`, `json`, а также классы `AbstractProvider`, `CreateResult` и `Messages` из соответствующих файлов.
2. **Определение списка моделей**: Создается словарь `models`, содержащий информацию о поддерживаемых моделях, таких как `gpt-3.5-turbo` и `gpt-4`, с их идентификаторами и именами.
3. **Определение класса `Aivvm`**:
   - Указывается URL сервиса (`https://chat.aivvm.com`).
   - Поддерживается потоковая передача (`supports_stream = True`).
   - Указывается, что провайдер поддерживает модели `gpt-3.5-turbo` и `gpt-4`.
4. **Реализация метода `create_completion`**:
   - Принимает параметры: `model` (строка, определяющая модель), `messages` (список сообщений), `stream` (логическое значение, определяющее, использовать ли потоковую передачу) и `kwargs` (дополнительные аргументы).
   - Если `model` не указана, используется `gpt-3.5-turbo` по умолчанию.
   - Проверяется, поддерживается ли указанная модель. Если нет, вызывается исключение `ValueError`.
   - Формируется словарь `json_data`, который содержит информацию о модели, сообщениях, API-ключе и дополнительных параметрах, таких как `system_message` и `temperature`.
   - Преобразуется словарь `json_data` в JSON-строку.
   - Определяются заголовки HTTP-запроса, включая `Content-Type`, `Content-Length`, `User-Agent` и другие.
   - Отправляется POST-запрос к `https://chat.aivvm.com/api/chat` с указанными заголовками и данными.
   - Обрабатывается ответ от сервера с использованием потоковой передачи.
   - Каждый чанк ответа декодируется в UTF-8 и возвращается через `yield`. Если возникает ошибка `UnicodeDecodeError`, используется кодировка `unicode-escape`.

Пример использования
-------------------------

```python
from src.endpoints.gpt4free.g4f.Provider.deprecated.Aivvm import Aivvm
from src.typing import Messages

# Создание списка сообщений для отправки
messages: Messages = [
    {"role": "user", "content": "Привет, как дела?"}
]

# Вызов метода create_completion для получения ответа от модели gpt-3.5-turbo
try:
    for chunk in Aivvm.create_completion(model="gpt-3.5-turbo", messages=messages, stream=True):
        print(chunk, end="")  # Вывод чанков ответа в консоль
except Exception as e:
    print(f"Произошла ошибка: {e}")
