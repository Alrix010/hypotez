## Как использовать класс `MetaAI`
=========================================================================================

Описание
-------------------------
Класс `MetaAI` реализует асинхронный генератор для взаимодействия с API Meta AI. Он позволяет отправлять запросы на модели Meta AI и получать ответы в виде асинхронного генератора.

Шаги выполнения
-------------------------
1. **Инициализация**:  Создайте объект класса `MetaAI` с опциональными параметрами `proxy` (прокси-сервер) и `connector` (настройка HTTP-соединения).
2. **Асинхронный генератор**: Используйте метод `create_async_generator` для создания асинхронного генератора, который будет генерировать ответы от модели Meta AI. Передайте в метод необходимую модель (`model`), сообщения (`messages`) и опциональный параметр `proxy`.
3. **Итерация по ответам**: Итерируйтесь по результатам асинхронного генератора, используя цикл `async for`. Каждый шаг цикла будет возвращать частичный или полный ответ от модели.
4. **Обработка ответов**: Обработайте полученные ответы, например, отобразите их на экране или сохраните в файл.

Пример использования
-------------------------

```python
from hypotez.src.endpoints.gpt4free.g4f.Provider.needs_auth.MetaAI import MetaAI

async def main():
    """Пример использования класса MetaAI."""

    # Создаем объект MetaAI
    meta_ai = MetaAI()

    # Формируем сообщения для запроса
    messages = [
        {"role": "user", "content": "Привет, как дела?"},
    ]

    # Создаем асинхронный генератор
    async for chunk in await MetaAI.create_async_generator(model="meta-ai", messages=messages):
        print(chunk)

if __name__ == "__main__":
    import asyncio
    asyncio.run(main())
```
