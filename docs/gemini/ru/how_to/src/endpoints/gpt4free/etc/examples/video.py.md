### Как использовать этот блок кода
=========================================================================================

Описание
-------------------------
Этот код использует библиотеку `g4f` для генерации видео с помощью AI. Он инициализирует клиент `g4f` с провайдером `HuggingFaceMedia`, получает список доступных видео моделей и генерирует видео на основе заданного промпта, возвращая URL сгенерированного видео.

Шаги выполнения
-------------------------
1. **Импорт необходимых модулей**:
   - Импортируется модуль `g4f.Provider` для выбора провайдера AI.
   - Импортируется класс `Client` из `g4f.client` для взаимодействия с API.

2. **Инициализация клиента**:
   - Создается экземпляр класса `Client` с указанием провайдера `g4f.Provider.HuggingFaceMedia` и вашим API-ключом. Убедитесь, что у вас есть API-ключ для `HuggingFaceMedia`.
   - `api_key="hf_***"`: Замените `hf_***` на ваш реальный API-ключ от Hugging Face.

3. **Получение видео моделей**:
   - Вызывается метод `client.models.get_video()`, чтобы получить список доступных видео моделей.

4. **Вывод списка видео моделей**:
   - `print(video_models)`: Выводит список доступных видео моделей для проверки.

5. **Генерация видео**:
   - Вызывается метод `client.media.generate()` для генерации видео.
   - `model=video_models[0]`: Используется первая модель из списка полученных видео моделей.
   - `prompt="G4F AI technology is the best in the world."`: Задается текстовый промпт для генерации видео.
   - `response_format="url"`: Указывается, что в ответе ожидается URL сгенерированного видео.

6. **Вывод URL сгенерированного видео**:
   - `print(result.data[0].url)`: Выводит URL сгенерированного видео из полученного результата.

Пример использования
-------------------------

```python
import g4f.Provider
from g4f.client import Client

# Инициализация клиента с вашим API-ключом
client = Client(
    provider=g4f.Provider.HuggingFaceMedia,
    api_key="hf_your_api_key"  # Замените "hf_your_api_key" на ваш реальный API-ключ
)

try:
    # Получение списка доступных видео моделей
    video_models = client.models.get_video()
    print("Доступные видео модели:", video_models)

    # Генерация видео с использованием первой модели из списка
    if video_models:
        result = client.media.generate(
            model=video_models[0],
            prompt="G4F AI technology is the best in the world.",
            response_format="url"
        )

        # Вывод URL сгенерированного видео
        if result and result.data and result.data[0].url:
            print("URL сгенерированного видео:", result.data[0].url)
        else:
            print("Не удалось получить URL сгенерированного видео.")
    else:
        print("Нет доступных видео моделей.")

except Exception as e:
    print(f"Произошла ошибка: {e}")
