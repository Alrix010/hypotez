# Модуль настройки g4f

## Обзор

Модуль `setup.py` предназначен для настройки и установки пакета `g4f`. Он определяет метаданные пакета, зависимости и точки входа.

## Подробней

Модуль использует библиотеку `setuptools` для определения параметров установки, таких как имя пакета, версия, автор, описание, зависимости и классификаторы. Он также включает данные пакета, такие как файлы интерфейса и моделей.

## Переменные

*   `here` (str): Абсолютный путь к каталогу, содержащему файл `setup.py`.
*   `long_description` (str): Длинное описание пакета, прочитанное из файла `README.md`.
*   `INSTALL_REQUIRE` (list[str]): Список основных зависимостей, необходимых для установки пакета.
*   `EXTRA_REQUIRE` (dict[str, list[str]]): Словарь дополнительных зависимостей, разделенных по категориям (например, `all`, `slim`, `image`, `webview`, `api`, `gui`, `search`, `local`, `files`).
*   `DESCRIPTION` (str): Краткое описание пакета.

## Функции

### `setup`

**Назначение**: Функция для настройки и установки пакета `g4f`.

**Параметры**:

*   `name` (str): Имя пакета (значение: `'g4f'`).
*   `version` (str): Версия пакета (значение: `os.environ.get("G4F_VERSION")`).
*   `author` (str): Автор пакета (значение: `'Tekky'`).
*   `author_email` (str): Электронная почта автора (значение: `'<support@g4f.ai>'`).
*   `description` (str): Краткое описание пакета (значение: `DESCRIPTION`).
*   `long_description_content_type` (str): Тип контента для длинного описания (значение: `'text/markdown'`).
*   `long_description` (str): Длинное описание пакета (значение: `long_description`).
*   `packages` (list[str]): Список пакетов, которые будут включены (значение: `find_packages()`).
*   `package_data` (dict[str, list[str]]): Дополнительные файлы, которые будут включены в пакет (например, файлы интерфейса и моделей).
*   `include_package_data` (bool): Указывает, следует ли включать данные пакета (значение: `True`).
*   `install_requires` (list[str]): Список основных зависимостей (значение: `INSTALL_REQUIRE`).
*   `extras_require` (dict[str, list[str]]): Словарь дополнительных зависимостей (значение: `EXTRA_REQUIRE`).
*   `entry_points` (dict[str, list[str]]): Точки входа для консольных скриптов (значение: `{'console_scripts': ['g4f=g4f.cli:main']}`).
*   `url` (str): URL-адрес репозитория (значение: `'https://github.com/xtekky/gpt4free'`).
*   `project_urls` (dict[str, str]): Дополнительные URL-адреса проекта (например, для отслеживания ошибок).
*   `keywords` (list[str]): Список ключевых слов, связанных с пакетом.
*   `classifiers` (list[str]): Список классификаторов, описывающих пакет.

**Как работает функция**:

1.  Читает длинное описание из файла `README.md`.
2.  Определяет основные и дополнительные зависимости.
3.  Настраивает пакет с использованием указанных параметров.

**Примеры**:

Для установки пакета `g4f` необходимо выполнить команду:

```bash
python setup.py install
```

Для установки дополнительных зависимостей:

```bash
pip install g4f[all]
```
