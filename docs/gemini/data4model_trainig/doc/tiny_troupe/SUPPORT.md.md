# Поддержка

## Как сообщать о проблемах и получать помощь

В этом проекте используются GitHub Issues для отслеживания ошибок и запросов на добавление функций. Пожалуйста, поищите существующие проблемы, прежде чем создавать новые, чтобы избежать дублирования. Для новых проблем подайте свой запрос об ошибке или функции в виде нового Issue.

Для получения помощи и ответов на вопросы об использовании этого проекта, пожалуйста, обратитесь к нашей Вики или напишите сообщение на нашем форуме обсуждений, оба здесь, на GitHub.
