# Модуль `edit_campaign.py`

## Обзор

Модуль предназначен для редактирования рекламных кампаний на AliExpress. Он предоставляет функциональность для обработки и изменения параметров кампаний, а также категорий кампаний.

## Подробнее

Модуль `edit_campaign.py` является частью проекта `hypotez` и используется для автоматизации процесса управления рекламными кампаниями на платформе AliExpress. Он включает в себя классы и функции для редактирования кампаний, обработки категорий и общей настройки. Расположение файла в структуре проекта указывает на его принадлежность к подсистеме управления кампаниями поставщиков (suppliers), в частности, AliExpress.
