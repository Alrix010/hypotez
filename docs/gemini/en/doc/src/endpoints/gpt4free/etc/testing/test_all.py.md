# Модуль тестирования всех моделей

## Обзор

Этот модуль предназначен для тестирования работоспособности различных моделей GPT-3.5 и GPT-4 с использованием библиотеки `g4f`. Он содержит функции для асинхронного тестирования моделей и проверки их доступности.

## Детали

Модуль тестирует заданный набор моделей, отправляя им запрос на генерацию стихотворения о дереве и проверяя, возвращают ли они корректный ответ. Результаты тестирования выводятся в консоль.

## Классы

В данном модуле классы отсутствуют.

## Функции

### `test`

```python
async def test(model: g4f.Model) -> bool:
    """Тестирует заданную модель на работоспособность.

    Args:
        model (g4f.Model): Модель для тестирования.

    Returns:
        bool: `True`, если модель работает корректно, `False` в противном случае.

    Raises:
        Exception: Если во время тестирования модели возникает исключение.
    
    Как работает функция:
    - Функция пытается сгенерировать стихотворение о дереве с использованием заданной модели.
    - Сначала делается попытка асинхронного создания чата, а затем обработки результата в потоковом режиме.
    - Если возникает исключение, делается попытка выполнить то же самое асинхронно.
    - В случае успеха функция возвращает `True`, иначе выводит информацию об ошибке и возвращает `False`.
    """
```

### `start_test`

```python
async def start_test() -> None:
    """Запускает тестирование заданных моделей и выводит результаты.
    Args:
        model (g4f.Model): Модель для тестирования.

    Returns:
        None

    Как работает функция:
    - Определяется список моделей для тестирования (`models_to_test`), включающий `gpt-3.5-turbo` и `gpt-4`.
    - Создается пустой список `models_working` для хранения названий работающих моделей.
    - Для каждой модели из списка `models_to_test` вызывается функция `test`, чтобы проверить её работоспособность.
    - Если модель работает корректно, её название добавляется в список `models_working`.
    - В конце выводится список работающих моделей.
    """
```

## Запуск тестов

```python
asyncio.run(start_test())
```

Запускает асинхронную функцию `start_test`, которая выполняет тестирование моделей и выводит результаты.

## Примеры
Примеры использования функций не требуются, поскольку `start_test` запускается автоматически при запуске модуля.
