# Модуль `utils`

## Обзор

Модуль `utils` предоставляет набор утилитных функций для работы с куки-файлами браузеров. 

## Детали

Этот модуль используется для получения куки-файлов из различных браузеров, таких как Chrome, Safari, Firefox, Edge, Opera, Brave, Opera GX, Vivaldi. 

## Классы

### `Utils`

**Описание**: Класс `Utils` содержит список поддерживаемых браузеров и функцию для получения куки-файлов.

**Атрибуты**:
- `browsers`: Список поддерживаемых браузеров (Chrome, Safari, Firefox, Edge, Opera, Brave, Opera GX, Vivaldi).

**Методы**:
- `get_cookies(domain: str, setName: str = None, setBrowser: str = False) -> dict`: Функция для получения куки-файлов из указанного домена.

## Функции

### `get_cookies`

**Цель**: Извлечение куки-файлов из указанного домена.

**Параметры**:
- `domain` (str): Домен, из которого необходимо получить куки-файлы.
- `setName` (str, optional): Имя конкретного куки-файла, который требуется получить. По умолчанию `None`.
- `setBrowser` (str, optional): Имя браузера, из которого необходимо получить куки-файлы. По умолчанию `False`.

**Возвращает**:
- `dict`: Словарь с куки-файлами.

**Возможные исключения**:
- `ValueError`: Если указанный куки-файл `setName` не найден в любом из браузеров.

**Как работает функция**:

- Функция `get_cookies`  итеративно проходит по списку поддерживаемых браузеров.
- Для каждого браузера функция пытается получить куки-файлы из указанного домена.
- Если функция успешно получает куки-файлы из браузера, она добавляет их в словарь `cookies`.
- Если задано имя куки-файла `setName`, функция возвращает только этот куки-файл.
- Если не задано имя куки-файла, функция возвращает словарь всех полученных куки-файлов.

**Примеры**:

```python
# Получение всех куки-файлов с сайта example.com
cookies = Utils.get_cookies('example.com')

# Получение куки-файла с именем 'session_id' с сайта example.com
session_cookie = Utils.get_cookies('example.com', setName='session_id')

# Получение куки-файлов из браузера Chrome
chrome_cookies = Utils.get_cookies('example.com', setBrowser='chrome')
```
