```
# Описание инструкций для бота

## Обзор

Этот документ содержит описание инструкций для бота, используемых для взаимодействия с моделью машинного обучения. Каждая инструкция представляет собой команду, которую пользователь может отправить боту для выполнения определенной задачи, такой как обучение модели, тестирование модели, архивирование файлов и т. д.

## Инструкции

### `!hi`

**Назначение**: Приветствует пользователя.

**Описание**: Команда `!hi` предназначена для приветствия пользователя и начала взаимодействия с ботом.

**Пример использования**:

```
!hi
```

### `!train <data> <data_dir> <positive> <attachment>`

**Назначение**: Обучает модель с предоставленными данными.

**Описание**: Команда `!train` используется для обучения модели машинного обучения. В зависимости от типа данных, можно использовать разные параметры:

-   `<data>`: Путь к файлу с данными для обучения.
-   `<data_dir>`: Путь к директории, содержащей данные для обучения.
-   `<positive>`: Дополнительный параметр (описание отсутствует).
-   `<attachment>`: Файл, прикрепленный к сообщению, содержащий данные для обучения.

**Пример использования**:

```
!train data.csv data_dir positive_value attachment.zip
```

### `!test <test_data>`

**Назначение**: Тестирует модель с предоставленными JSON данными.

**Описание**: Команда `!test` используется для тестирования обученной модели с использованием предоставленных данных в формате JSON.

-   `<test_data>`: Путь к файлу с данными для тестирования в формате JSON.

**Пример использования**:

```
!test test_data.json
```

### `!archive <directory>`

**Назначение**: Архивирует файлы в указанной директории.

**Описание**: Команда `!archive` используется для создания архива файлов, находящихся в указанной директории.

-   `<directory>`: Путь к директории, которую необходимо заархивировать.

**Пример использования**:

```
!archive /path/to/directory
```

### `!select_dataset <path_to_dir_positive> <positive>`

**Назначение**: Выбирает набор данных для обучения из указанной директории.

**Описание**: Команда `!select_dataset` используется для выбора набора данных для обучения модели из указанной директории.

-   `<path_to_dir_positive>`: Путь к директории, содержащей положительные примеры для обучения.
-   `<positive>`: Дополнительный параметр (описание отсутствует).

**Пример использования**:

```
!select_dataset /path/to/positive/examples positive_value
```

### `!instruction`

**Назначение**: Отображает сообщение с инструкциями.

**Описание**: Команда `!instruction` используется для отображения списка доступных команд и их описания.

**Пример использования**:

```
!instruction
