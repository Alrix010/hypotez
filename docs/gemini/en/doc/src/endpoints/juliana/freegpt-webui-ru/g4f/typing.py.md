# Модуль типизации для g4f

## Обзор

Данный модуль предоставляет типы данных, используемые в модуле `g4f`. Он определяет типы, которые используются для представления различных типов данных, таких как хеш-сумма SHA-256, запросы и ответы.

## Детали

Модуль используется для повышения четкости и безопасности кода, позволяя компилятору проверять соответствие типов данных при работе с ними.

## Типы данных

### `sha256`

**Описание**: Тип данных для представления хеш-суммы SHA-256.

**Примеры**:

```python
from g4f.typing import sha256

hash: sha256 = 'a94a8fe5ccb19ba61c4c0873d391e987982fbbd30870803048f2d73b60c01fc3'
```
